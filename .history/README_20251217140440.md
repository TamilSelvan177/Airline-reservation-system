# Airline Reservation System (Java + JDBC)

A console-based Airline Reservation System developed using **Java**, **JDBC**, and **Microsoft SQL Server**, following core **Object-Oriented Programming (OOPS)** principles.

This project demonstrates how a real-world reservation system can be designed with persistent storage, proper abstraction, and clean separation of concerns.
