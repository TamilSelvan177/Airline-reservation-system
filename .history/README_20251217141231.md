# Airline Reservation System

A console-based Airline Reservation System built using:

- Java
- JDBC
- SQL Server
- OOPS concepts

## Features
- Book and cancel tickets
- Persistent passenger data
- Persistent seat management
- Revenue calculation
- SQL Server integration

## OOPS Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation

## How to Run
1. Create database and tables in SQL Server
2. Update JDBC URL if needed
3. Run AirlineReservationSystem.java
